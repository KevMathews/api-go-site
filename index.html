Introduction

Welcome to API-Go

The dev package designed to help jump start your React website!



API-Go was created for one simple reason, to ease the use of APIs!
Anyone from an aspiring student, to a senior level dev can benefit from this package.

No more squandering precious dev time studying up on how to interact with a particular API.

With API-Go that information, with examples, will be at your fingertips for easy reference!

So, how exactly does API-Go save you time?
We accomplish this in 2 simple ways:

  • By presenting you a simplified 'cheat-sheet' of the endpoints and parameters for the API of your choice.

  • By providing examples of fetch calls, and showing you the data in json format that is returned.

Essentially we packaged together the information needed to get started with some commonly used Free API's,
and present the info to you in an easy to read and access format.






Usage

To get started you can download API-Go from either GitHub or via our website.
Alternatively, you can also clone the directory from Github.
- We will be deploying soon via NPM -

Once the files are located on your computer CD into the folder and run 'npm i -g' to install
our package globally on your computer.
Take note that some installations might require sudo or administration access

From there you want to create a directory to launch API-go and your project from.
*Make sure the directory you launch api-go in ISN'T already a git hub repo already *

Once inside the directory type api-go from your terminal command line

You will be presented with several questions while navigating on the command line

  • You will be asked which API you want to work with?

  • If you have an API key for it yet?

    • If you do paste into into the command line when prompted and we will automatically insert it in the code for you when presenting ypou with examples and fetches.

    Next you need to name your project so we can properly create the directory structure for you.

    After that is all completed give the process up to a minute or so to download and install the packages and node modules needed to get you up and running.

    Once completed cd into the newly created directory that will contain all of the dynamically created files for you and type npm start to alternatively open VSCode or similar code editor open the imbedded terminal and type npm start

    What you will have is essentially a 'cheat sheet' at your finger tips to be able to reference query params and have example fetches to help get your jump started in your development.

    KEY to our design was including the examples in a navbar tab for you. The idea behind it is you can fully develop your react site leaving the nav bar component in place to reference the examples quickly as needed. when you are done needing the use of the examples its as simple as removing the nav bar component from the router.

    News API and using their Everything Endpoint

    News API allows you to search through over 75,000 news sources and blogs dating back the last 3 years.
    They have 2 main endpoints to query(everything and top-headlines) with many parameters for each

    The everything endpoint searches are ideal for news analysis and article discovery

    News API everything endpoint Query Parameters:


    PARAMETERS DESCRIPTION
    apiKey Your API key. Alternatively you can provide this via the X-Api-Key HTTP header.
    q Keywords or phrases to search for in the article title and body.
    sources A comma-seperated string of identifiers (maximum 20) for the news sources or blogs you want headlines from.
    domains A comma-seperated string of domains (mashable.com, techcrunch.com, engadget.com) to restrict the search to.
    from A date and optional time for the oldest article allowed. This should be in ISO 8601 format (e.g. 2021-04-03 or 2021-04-03T15:46:09)
    to A date and optional time for the newest article allowed. This should be in ISO 8601 format (e.g. 2021-04-03 or 2021-04-03T15:46:09)
    language The 2-letter ISO-639-1 code of the language you want to get headlines for. Possible options: ar de en es fr he it nl no pt ru se ud zh


    Sample News API Everything Endpoint Queries:

    https://newsapi.org/v2/everything?q=wandavision&apiKey=NEWS-API-KEY

    Searches all sources for articles on Wandavision. An example of 1 object returned is below:



    https://newsapi.org/v2/everything?q=tesla&from=2021-04-01&to=2021-04-01&sortBy=popularity&apiKey=NEWS-API-KEY

    Searches all sources for articles on Tesla from 2021-04-01 and sorted by populatity An example of 1 object returned is below:



    https://newsapi.org/v2/everything?domains=engadget.com,mashable.com&apiKey=NEWS-API-KEY

    Searches all sources for articles by Engadget and Mashable. Example of 1 object returned is below:
